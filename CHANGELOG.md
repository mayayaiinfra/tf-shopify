# Changelog

All notable changes to the THUNDERFIRE Shopify App will be documented in this file.

## [1.0.0] - 2026-02-21

### Added
- **Fleet Dashboard** - View all connected THUNDERFIRE nodes
- **Node Detail** - 7-field CHITRAL health breakdown
- **Product Binding** - Link Shopify products to monitoring nodes
- **Auto-Goals** - Send THETA goals when orders are placed
- **NOP Marketplace** - Browse and track service contracts
- **Storefront Widget** - Theme App Extension for product pages
- **Alerts** - Health degradation notifications
- **Settings** - API configuration with connection test
- **Billing** - Shopify Billing API integration (Free/Pro tiers)
- **GDPR Webhooks** - Customer data request/erasure handlers
- **Vitest Tests** - 59 tests covering all routes

### Security
- HMAC signature validation on webhooks
- OAuth 2.0 authentication flow
- CSP headers enabled
